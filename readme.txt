== Description ==

It removes advertising from your backend. Many times, on your request.

This plugin is supported by a community of people who report the plugins that add advertising to the backend.

Many times you activate this plugin and still see spam and advertising. In all those cases, you have to open a thread on the support forum and report the plugin that is adding spam.

We will do the rest and inform you when a new version is available to hide or remove the spam that you reported.



== Help ==

If you see spam or advertising, don't hesitate to open a thread on the <a href="https://wordpress.org/support/plugin/no-spam-in-backend/">support forum</a>
